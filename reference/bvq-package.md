# bvq: Barcelona Vocabulary Questionnaire Database and Helper Functions

Download, clean, and process the Barcelona Vocabulary Questionnaire
(BVQ) data. BVQ is a vocabulary inventory developed for assesing the
vocabulary of Catalan-Spanish bilinguals infants from the Metropolitan
Area of Barcelona (Spain). This package includes functions to download
the data from formr servers, and return the processed data in multiple
formats.
